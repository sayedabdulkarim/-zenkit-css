Get started with ZenKit

ZenKit is a powerful, feature-packed frontend toolkit with 100% Bootstrap 5.3 feature parity. Build anything from prototype to production.

Quick start

Get started by including ZenKit's production-ready CSS and JavaScript via CDN without any build steps.

<!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>ZenKit demo</title>
    <link rel="stylesheet" href="path/to/zenkit.min.css">
  </head>
  <body>
    <h1>Hello, world!</h1>
    <script src="path/to/zenkit.umd.js"></script>
  </body>
</html>

Download ZenKit to get the compiled CSS and JavaScript, source code, or include it with npm.

JavaScript

Bring ZenKit to life with our JavaScript plugins. Learn about each plugin and our data and programmatic API options.

Individual or compiled

Plugins can be included individually or all at once using zenkit.umd.js.

// ES Modules
import { Modal, Dropdown, Toast } from './js/zenkit.js';

// UMD (browser global)
<script src="zenkit.umd.js"></script>
<script>
  const modal = new ZenKit.Modal('#myModal');
</script>

Data attributes

Nearly all plugins can be enabled via data attributes. You can enable with data-toggle.

<button data-toggle="modal" data-target="#myModal">Launch modal</button>

Animation Control

Control iteration, direction, and play state:

// Iteration
.animate-once     { animation-iteration-count: 1; }
.animate-twice    { animation-iteration-count: 2; }
.animate-infinite { animation-iteration-count: infinite; }

// Direction
.animate-normal            { animation-direction: normal; }
.animate-reverse           { animation-direction: reverse; }
.animate-alternate         { animation-direction: alternate; }
.animate-alternate-reverse { animation-direction: alternate-reverse; }

// Fill Mode
.animate-fill-none      { animation-fill-mode: none; }
.animate-fill-forwards  { animation-fill-mode: forwards; }
.animate-fill-backwards { animation-fill-mode: backwards; }
.animate-fill-both      { animation-fill-mode: both; }

// Play State
.animate-running { animation-play-state: running; }
.animate-paused  { animation-play-state: paused; }
.hover\:animate-paused:hover { animation-play-state: paused; }
